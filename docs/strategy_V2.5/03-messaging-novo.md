# Novo Framework de Messaging ‚Äî Copy para Landing Page

> Regra: Problema antes de solucao. Beneficio antes de feature. Especifico antes de generico.
> Audience primaria: Donos de agencias de marketing digital (2-15 funcionarios, BR).

---

## Principios do Novo Copy

1. **Problema-primeiro:** O usuario reconhece a dor antes de ver o produto
2. **Linguagem do cliente:** Usar as palavras que o ICP usa (nao nosso jargao tecnico)
3. **Especificidade:** Numeros reais, exemplos concretos, verticais especificas
4. **Show, don't tell:** Demo interativo vale mais que 1000 palavras de feature list
5. **Urgencia real:** Por que resolver isso AGORA? (custo de inacao)
